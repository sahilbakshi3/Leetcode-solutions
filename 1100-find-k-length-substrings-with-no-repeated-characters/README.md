<h2><a href="https://leetcode.com/problems/find-k-length-substrings-with-no-repeated-characters/">1100. Find K-Length Substrings With No Repeated Characters</a></h2><h3>Medium</h3><hr><p>Given a string <code>s</code> and an integer <code>k</code>, return <em>the number of substrings of length <code>k</code> with no repeated characters</em>.</p>

<p>A substring is a contiguous sequence of characters within a string.</p>

<p>&nbsp;</p>

<p><strong class="example">Example 1:</strong></p>

<pre>
<strong>Input:</strong> s = "havefunonleetcode", k = 5
<strong>Output:</strong> 6
<strong>Explanation:</strong>
The valid substrings are:
"havef",
"avefu",
"vefun",
"efuno",
"etcod",
"tcode"
</pre>

<p><strong class="example">Example 2:</strong></p>

<pre>
<strong>Input:</strong> s = "home", k = 5
<strong>Output:</strong> 0
</pre>

<p>&nbsp;</p>

<p><strong>Constraints:</strong></p>

<ul>
	<li><code>1 &lt;= s.length &lt;= 10^4</code></li>
	<li><code>1 &lt;= k &lt;= 10^4</code></li>
	<li><code>s</code> consists of lowercase English letters.</li>
</ul>
