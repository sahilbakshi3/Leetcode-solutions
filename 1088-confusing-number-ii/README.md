<h2><a href="https://leetcode.com/problems/confusing-number-ii/">1088. Confusing Number II</a></h2><h3>Hard</h3><hr><p>A <strong>confusing number</strong> is a number that when rotated <code>180</code> degrees becomes a different number with each digit valid.</p>

<p>The valid digits after rotation are:</p>

<ul>
	<li><code>0 → 0</code></li>
	<li><code>1 → 1</code></li>
	<li><code>6 → 9</code></li>
	<li><code>8 → 8</code></li>
	<li><code>9 → 6</code></li>
</ul>

<p>Digits <code>2,3,4,5,7</code> are invalid.</p>

<p>Return <em>the number of confusing numbers in the range</em> <code>[1, n]</code>.</p>

<p>&nbsp;</p>

<p><strong class="example">Example 1:</strong></p>

<pre>
<strong>Input:</strong> n = 20
<strong>Output:</strong> 6
<strong>Explanation:</strong>
The confusing numbers are:
6, 9, 10, 16, 18, 19
</pre>

<p><strong class="example">Example 2:</strong></p>

<pre>
<strong>Input:</strong> n = 100
<strong>Output:</strong> 19
</pre>

<p>&nbsp;</p>

<p><strong>Constraints:</strong></p>

<ul>
	<li><code>1 &lt;= n &lt;= 10^9</code></li>
</ul>
