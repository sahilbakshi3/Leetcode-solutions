<h2><a href="https://leetcode.com/problems/largest-unique-number/">1133. Largest Unique Number</a></h2><h3>Easy</h3><hr><p>Given an integer array <code>nums</code>, return <em>the largest integer that occurs exactly once</em>. If no integer occurs exactly once, return <code>-1</code>.</p>

<p>&nbsp;</p>

<p><strong class="example">Example 1:</strong></p>

<pre>
<strong>Input:</strong> nums = [5,7,3,9,4,9,8,3,1]
<strong>Output:</strong> 8
<strong>Explanation:</strong>
The unique numbers are [5,7,4,8,1].
The largest among them is 8.
</pre>

<p><strong class="example">Example 2:</strong></p>

<pre>
<strong>Input:</strong> nums = [9,9,8,8]
<strong>Output:</strong> -1
<strong>Explanation:</strong>
There is no number that appears exactly once.
</pre>

<p>&nbsp;</p>

<p><strong>Constraints:</strong></p>

<ul>
	<li><code>1 &lt;= nums.length &lt;= 2000</code></li>
	<li><code>0 &lt;= nums[i] &lt;= 1000</code></li>
</ul>
